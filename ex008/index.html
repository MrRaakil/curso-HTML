Principais formatações

Negrito / DESTAQUE

Nesse frase temos um termo em negrito usando a tag <b> (não semantica)

Nesta frase, tenho um termo de destaque usanda a tag <strong> (semantica)

Itálico / ÊNFASE

Já nesta frase, temos um termo em itálico usando a tag <i> (não semantico)

Já nesta frase, temos um termo em itálico usando ctrl + shift + p

Nesta frase, temos um termo em ênfase usando a tag <em> (semantica)

Texto marcado

Podemos criar um texto marcado ultilizando a tag <mark>

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo

Texto deletado/riscado

Podemos marcar um texto como excluído/riscado para indicar que ele deve ser lido, mas não considerado usando a tag <del> (semantico)

Texto inserido

Podemos marcar um texto como inserido/sublinhado para dar uma ênfase e indicar que ele foi adicionado depois usando a tag <ins> (semantico)

Também é possível sublinhar ultilizando a tag <u> (não semantico)

Texto sobrescrito

Para inserir coisas do tipo x2+3 ultilixa a tag <sup> (coisas elevadas)

Texto subscrito

Para inserir coisas do tipo H2O <sub>
